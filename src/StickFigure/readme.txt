初期設定
	.NET Framework 4.7.2をインストールする

使い方
	・棒人間が表示されている領域からドラッグ＆ドロップでSD-webuiに画像をドロップできる

	・棒人間の各ポイントをマウスで移動可能
	　・マウスの右ボタンなら子ポイントも同時に移動
	　・マウスの中ボタンならそのポイントのみ移動
	　・Bodyのポイントを移動するとすべてのポイントが移動する

	・Presetの名前入力＆Save、Loadで現在の状態を保存、ロードできる
	
	・ポイントをクリック後、Scaling、Rotationで棒人間の拡大や回転ができる

	・New Figureで棒人間追加

	・Delete Figureで棒人間削除

	・右側のツリーで関節の点を描画するかを制御できる
	　関節が明るいと正面側、暗いと裏側と判断されている？

設定値
	Width,Heigh: 画像の大きさ
	Magin:       編集領域の余白
	Line Width:  棒人間の太さ
	Show Label:  手足のラベルを画面上に表示
	Draw Eys:    画像生成時に目のラインを描かない

その他
